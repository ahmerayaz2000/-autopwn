AutoPwn

A custom-built Python tool that runs 5 security modules in parallel, aggregates every finding, scores it by CVSSv3 severity, and generates a professional PDF report, all from a single URL.

01

Reconnaissance

DNS enumeration, WHOIS lookup, zone-transfer vulnerability check, reverse DNS.

02

Port Scanning

Nmap-powered scan of top 1000 ports. Flags RDP, SMB, Redis, MongoDB and 16 other high-risk services.

03

Web Vulnerability Scan

Tests for SQL injection, open redirects, exposed .env / .git files, admin panels, and 25+ sensitive paths.

04

HTTP Header Audit

Checks for HSTS, CSP, X-Frame-Options, X-Content-Type-Options and server technology disclosure.

05

SSL / TLS Inspection

Certificate expiry, self-signed detection, deprecated TLS 1.0/1.1 protocol support.
